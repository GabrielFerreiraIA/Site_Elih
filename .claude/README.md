### Exemplo 1: Criar Hero Section
```
Step 1: @brand-voice
"Define visual identity para Elih Seguros"

Step 2: @frontend-design
"Crie hero section com:
- Headline impactante
- Subtítulo
- CTA button
- Background image
- Responsivo"

Step 3: @design-system
"Use tokens para colors, spacing, fonts"

Step 4: @nextjs-turbopack
"Implemente em React/Next.js"

Step 5: @accessibility
"Valide WCAG AA compliance"

Step 6: @e2e-testing
"Crie testes"

Step 7: @seo
"Otimize meta tags"
```

### Exemplo 2: Design System Completo
```
@design-system
"Crie design system com:
- Paleta de cores (primária, secundária, neutros)
- Tipografia (headings, body)
- Spacing scale
- Componentes base (Button, Input, Card)
- Icons
- Variações (dark mode, states)

Baseado em:
- Tailwind CSS
- Modern design principles
- Acessibilidade WCAG AA"
```
